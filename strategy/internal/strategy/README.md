# 隔夜交易策略（Overnight Trading Strategy）

## 一、策略概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 策略名称 | 隔夜尾盘买入策略 |
| 策略类型 | T日尾盘买入 → T+1日卖出（隔夜持仓） |
| 选股频率 | 每日14:50执行一次 |
| 持仓周期 | 1个交易日（隔夜） |
| 最大持仓 | 4只股票 |
| 单只仓位 | 20%（占总资产） |
| 止损线 | -2%（基于买入价） |
| 止盈线 | +2%激活，回落1.5%触发 |
| 信号源 | 通达信(TDX)数据源 |

### 1.2 核心流程

```
T日 09:55  ──→ 大盘MACD检查（上证指数60分钟MACD红柱？）
                  │
                  否←──── 是
                  │       │
               不交易    T日 14:50  ──→ 全市场选股
                                          │
                                   风控五步流水线筛选
                                          │
                                   按排序结果买入（最多4只）
                                          │
                                    T+1日  ──→ 监控卖出条件
                                          │
                              ┌───────┼───────┐
                              │       │       │
                            止损    止盈   收盘卖出
```

## 二、大盘环境过滤

### 2.1 上证指数60分钟MACD

在 **09:55** 检查上证指数（000001）的60分钟K线MACD状态。

**计算步骤：**
1. 获取上证指数60分钟K线数据（截至当日首根60分K线）
2. 提取收盘价序列 `close[]`
3. 计算EMA12 = EMA(close, 12) — 12周期指数移动平均
4. 计算EMA26 = EMA(close, 26) — 26周期指数移动平均
5. DIF = EMA12 - EMA26
6. DEA = EMA(DIF, 9)
7. MACD Bar = (DIF - DEA) × 2

**判定：** MACD Bar > 0（红柱） → 允许当日交易
**数据要求：** 至少30根60分钟K线，否则视为数据不足，不交易

### 2.2 交易日判定

- 仅周一到周五为交易日
- 法定节假日需额外过滤（回测中已考虑）

## 三、选股规则（核心筛选）

两套条件 **XG1 ∩ XG2** 同时满足，再加上三项约束。全市场约5200只股票逐只检查。

### 3.1 公共变量（每只股票预先计算）

| 变量 | 计算方式 | 用途 |
|------|---------|------|
| `MA5` | 5日简单移动平均 | 趋势判断 |
| `MA10` | 10日简单移动平均 | 趋势判断 |
| `VOL5` | 5日平均成交量 | 量能基准 |
| `DIF` | EMA12 - EMA26 | MACD快线 |
| `DEA` | EMA(DIF, 9) | MACD慢线 |
| `MACD Bar` | (DIF - DEA) × 2 | MACD柱状图 |
| `K/D/J` | RSV的SMA计算 | KDJ指标 |
| `换手率` | 成交量 / 流通股本 × 100% | 活跃度 |
| `筹码获利` | 最近250日收盘价低于当前价的比例 | 筹码分布 |

### 3.2 XG1 — 主条件

```
1. 阳线:      Close > Open AND Close > PrevClose
2. MA5之上:   Close > MA5
3. 放量:      Volume > VOL5
              或者 换手率 5% ~ 15%
4. MACD多头:  MACD Bar > -0.7
              AND MACD Bar > Prev MACD Bar（柱放大）
              AND DIF >= Prev DIF - 0.02（DIF不下滑）
5. KDJ未超买: J < 88
6. 涨幅约束:
              换手率 ≤ 4%  → 涨幅 < 4.1%
              4% < 换手率 ≤ 5% → 涨幅 < 4.6%
              换手率 > 5%  → 涨幅 < 5.2%
7. 换手率 > 3.5%
8. 筹码获利比例 15% ~ 85%
9. 剔除KDJ高位见顶:
              特定4种J值形态 + 其中最大值 > 95
10. 剔除MACD特殊死叉:
              前3日DIF死叉DEA → 满足特定低位条件后豁免
```

### 3.3 XG2 — 辅助条件（任一满足即可）

```
1. MA5金叉MA10 且 D < 60
2. KDJ金叉 且 D < 60
3. DIF金叉DEA
4. KDJ隔日金叉 且 D < 60

附加约束:
  换手率 3.5% ~ 15%
  DIF > -1
```

### 3.4 约束1 — 剔除昨日涨停

| 板块 | 昨日涨幅≥阈值 | 说明 |
|------|-------------|------|
| 沪深主板 | 9.8% | 普通涨停板 |
| 创业板 | 19.5% | 20%涨跌幅 |
| 科创板 | 19.5% | 20%涨跌幅 |

昨涨停的股票次日高开概率大但买入性价比低，予以剔除。

### 3.5 约束2 — K线结构（前置阴线后连阳）

**恰好3连阳：** 往前第4根K线为阴线，之后连续3根阳线
**恰好4连阳：** 往前第5根K线为阴线，之后连续4根阳线

确保买入时处于短期上升启动阶段，而非追高。

### 3.6 最终通过条件

```go
passed := XG1 AND XG2 AND NOT(昨日涨停) AND (3连阳 OR 4连阳)
```

## 四、风控排序（五步流水线）

选股通过后，对每只股票执行五步风控流水线，计算总分并进行排序。

### 4.1 Step 0 — 前置计算

| 触发条件 | 处罚 |
|---------|------|
| 筹码获利 ≥ 80% | 一级降序（排名后置） |
| 筹码获利 ≥ 85% | 禁止第一重仓 |
| 筹码获利 ≥ 90% | 下调至🟡评级 |
| J > 80 | 脉冲仓位降至2-3% |
| J > 85 | 主线降备选 |
| J > 90 | 划入🟡评级 |
| 筹码≥80% 且 J>80%（双高危共振） | 仓位减半，高开预期压缩至0-1.5% |
| 周五 | 筹码档位-5%，KDJ阈值-5（周五折价） |

### 4.2 Step 1 — 板块+情绪修正

| 因子 | 影响 |
|------|------|
| 轮动题材 | 最高🟡评级（硬性红线） |
| 板块梯队（同板块≥2只入选） | +10～15分 |
| 板块主力净流入 > 1亿 | +15分 |
| 板块主力净流出 > 1亿 | -20分 |
| 板块涨停股 ≥ 3只 | +15分 |
| 涨停股 ≥ 1只 | +8分 |
| 价涨量缩 | -15分 |
| 放量尾盘拉升 | -10分 |
| 板块分歧（个股独涨板块普跌） | -15分 |

### 4.3 Step 2 — 技术定级 🟢🟡🔴

| 条件 | 评级 |
|------|------|
| 主线题材 + 阳线 + KDJ低位金叉 + MACD翻红 | 🟢 看涨 |
| 阳线 + MACD翻红（部分达标） | 🟡 震荡 |
| 其他（不达标） | 🔴 易跌 |
| MACD红柱持续放大（Bar>0.1 且递增） | 额外+10分 |

### 4.4 Step 3 — 量能强弱扣分

| 量能形态 | 扣分 |
|---------|------|
| 温和放量（V介于VOL5~VOL5×1.5） | 不扣分 |
| 单日爆量（V > VOL5×2） | 主线-10，轮动-25 |
| 涨停次日放量滞涨 | -30 |
| 缩量上涨 | -10 |
| 午后明显缩量（下午量/上午量 < 0.7） | -10 |

### 4.5 Step 4 — 预判+排序

| 情景 | 预判 |
|------|------|
| 主线题材无分歧 | 高开2-3%稳步冲高 |
| 主线题材有分歧 | 平开震荡，冲高易跳水 |
| 轮动前排 | 仅早盘半小时套利 |
| 轮动后排 | 冲高回落，无持续行情 |
| 多重高危共振（≥2项） | 仓位减半 |

**排序规则：**

1. **第一权重**: 隔夜低开风险 OvernightRisk（越低越好）
   - 筹码获利越高风险越大（≥80% +30）
   - J值越高风险越大（>80 +20）
   - 换手率异常 >15%（+15）
   - 收盘价在MA10下方（+15）
   - 周五（+10）
   - 大盘股 >100亿（+10）
   - 板块梯队降低风险（-15～-25）

2. **第二权重**: 早盘爆发力 PulseScore（越高越好）
   - 阳线（+10）
   - MACD翻红（+10）
   - J>80（-20）
   - 量能配合（+10）
   - 周五折价（-10）

3. **附加规则**: 流通市值>100亿的股票永久排在小盘股后方

### 4.6 最终建议

| 评级 | 建议 |
|------|------|
| 🟢 + 小盘 | 主仓重仓 |
| 🟢 + 大盘 | 主仓隔夜 |
| 🟡 + 无双风险 | 轻仓套利 |
| 🟡 + 双风险 | 超跌观察 |
| 🔴 | 规避 |

## 五、卖出规则

### 5.1 止损（条件1，优先级最高）

```
触发: 日内最低价 <= 买入价 × (1 - StopLossPct)
      默认 StopLossPct = 0.02（-2%）
卖出价: 买入价 × (1 - StopLossPct)
        若日内最低价更低，以更低价格成交（实际中更差情况）
```

### 5.2 止盈（条件2）

```
激活: 日内最高价 >= 买入价 × 1.02（涨幅达+2%）
机制: 从峰值回落，回落幅度 >= 1.5%（峰值 × 0.985）
去毛刺: 回落条件连续满足4根1分钟K线才确认触发
确认后: 以确认时的价格卖出
```

### 5.3 收盘卖出（条件3，兜底）

```
触发: 日内既未止损也未止盈
卖出价: 当日收盘价
```

### 5.4 盈亏统计

| 卖出原因 | 统计归类 |
|---------|---------|
| 止盈 | 止盈次数+1（winCount++） |
| 止损 | 止损次数+1（loseCount++） |
| 收盘卖出（卖出金额 > 买入成本） | 止盈次数+1 |
| 收盘卖出（卖出金额 ≤ 买入成本） | 止损次数+1 |

## 六、交易执行

### 6.1 买入计算

```
目标金额 = 总资产 × PositionPct（默认20%）
手续费   = 目标金额 × FeeRate（默认万分之一）
可用额   = 目标金额 - 手续费
股数     = floor(可用额 / 成交价)  // 向下取整到整手
实际金额 = 股数 × 成交价
实际费用 = 实际金额 × FeeRate
现金扣减 = 实际金额 + 实际费用
```

### 6.2 卖出计算

```
成交金额 = 卖出价 × 股数
手续费   = 成交金额 × FeeRate
净收入   = 成交金额 - 手续费
现金增加 = 净收入
盈亏     = 净收入 - 买入成本
```

### 6.3 每日净值

```
持仓市值 = Σ(各持仓股当日收盘价 × 股数)
总资产   = 现金 + 持仓市值
日收益   = (当日总资产 - 前日总资产) / 前日总资产
最大回撤 = max(1 - 当日总资产 / 阶段最高总资产)
```

## 九、关键发现

### 9.1 大盘环境至关重要

- **2023年全年MACD绿柱** → 0交易，0收益（策略空仓躲过大跌）
- **2024年前6个月MACD绿柱** → 6个月空仓，但随后抓住924行情
- MACD过滤器有效规避了下跌趋势中的交易损耗

### 9.2 仓位影响

从5%→20%仓位放大，收益按比例放大约4倍，最大回撤约放大2倍。收益率与仓位基本呈线性关系。

### 9.3 止损影响

- **5%止损**: 胜率~46%，但单笔亏损大
- **2%止损**: 胜率~40-45%略降，但单笔亏损大幅缩小，**总收益提升2倍以上**

### 9.4 策略瓶颈

- 高度依赖MACD红柱环境（震荡市或下跌市无法交易）
- 胜率仅~45%，靠盈亏比获利（止盈幅度 > 止损幅度）
- 单日回撤可达6-7%（当持仓4只同时止损时）
